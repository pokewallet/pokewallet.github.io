PokeWallet Tutorial

Learn how to export Pokémon files using PKHeX and understand how PokeWallet generates unique cryptographic keys from your Pokémon data.

Exporting Pokémon with PKHeX

1

Download and Open PKHeX

Get the latest version of PKHeX from the official repository and open your save file.

2

Select Your Pokémon

Click on the Pokémon in your party or boxes that you want to export.

3

Export as .pk* File

Right-click the Pokémon and select "Set" then "Export" or use the Export button to save as a .pk3 to .pk7 file depending on your game generation.

4

Save the File

Choose a location to save your Pokémon file. Remember where you saved it for the next step.

Note: PKHeX supports generations 3-7. Make sure to export in the correct format for your game:

  • .pk3 - Generation 3 (Ruby, Sapphire, Emerald, FireRed, LeafGreen)
  • .pk4 - Generation 4 (Diamond, Pearl, Platinum, HeartGold, SoulSilver)
  • .pk5 - Generation 5 (Black, White, Black 2, White 2)
  • .pk6 - Generation 6 (X, Y, Omega Ruby, Alpha Sapphire)
  • .pk7 - Generation 7 (Sun, Moon, Ultra Sun, Ultra Moon)

How PokeWallet Generation Works

Data Extraction

PokeWallet reads specific data from your Pokémon file:

  • Species ID - The numerical identifier of your Pokémon species
  • PID (Personality ID) - A unique 32-bit value that determines many of your Pokémon's characteristics
  • TID (Trainer ID) - Your trainer's public ID number
  • SID (Secret ID) - Your trainer's hidden ID number
  • OT (Original Trainer) - The name of the Pokémon's original trainer

Unique Key Creation

These values are combined into a unique string:

UniqueKey = Species-PID-TID-SID-OT

Cryptographic Hashing

The unique string is processed through SHA-256 hashing to create a private key:

PrivateKey = SHA-256(UniqueKey)

WIF Format Conversion

The private key is then converted to Wallet Import Format (WIF):

  1. Add a prefix byte (0x80 for mainnet)
  2. Add a compression flag (0x01)
  3. Calculate a checksum using double SHA-256
  4. Encode everything in Base58

Security Note: Each Pokémon creates a completely unique cryptographic key. Even the same Pokémon species with different PIDs or different trainers will generate entirely different keys.

Frequently Asked Questions

Is this secure?

Yes, the entire process happens locally in your browser. No data is sent to any server.

Can I use these keys for real cryptocurrency?

While the keys are cryptographically valid, this tool is primarily for educational and experimental purposes.

What if I have the same Pokémon with the same stats?

The PID (Personality ID) is almost always unique, ensuring different keys even for seemingly identical Pokémon.

Which Pokémon generations are supported?

Generations 3 through 7 are supported (.pk3 to .pk7 files).
